- This is a modified version of the bomberman with multiplayer feature.
- We use an additional API called Kryonet for the network.
- The res folder is for resources, such as sounds and images.
- To run the project, run GameManager.java.
- When you run the game, click on the frame to set focus to the frame so that you can use keyboard input.
- To select options, press ENTER.
- The UML folder includes a general diagram for the whole program and zoom-in parts.
- The presentation is a Prezi file. Run Prezi.exe to see the presentation.
- To test the multiplayer on the same computer, when you get to the client menu, just leave the IP blank and choose ENTER.
